# How to make a 1.12.2 eaglercraft server
## Step 1: Setting up the environment
Run the following command's to start the server up
```Shell
cd bungee
java -jar bungee.jar &
cd ..
cd server
java -jar server.jar
```
or
```shell
bash eaglerserver
```
